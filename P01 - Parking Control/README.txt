== PT-BR ==

Trabalhei brevemente nesse projeto simples de Spring Boot com a intenção de aprimorar minhas habilidades.
Basicamente, é uma aplicação que faz o controle de um estacionamento, podendo adicionar novas vagas, e as informações contidas nela (id da vaga, nome do dono da vaga, carro, placa, etc). Para testes na API utilizei o Postman, e o banco utilizado para a aplicação foi o PostgreSQL. A IDE utilizada foi o IntelliJ.

== EN-US ==

I've worked briefly in this project using Spring Boot in order to improve my skills.
Basically it's an application that does the parking control, being able to add new spots, and the information in it (spot id, the owner of the spot, car, license plate, etc). For tests in the API, i've decided to utilize Postman, and the database used for this application was PostgreSQL. The chosen integrated developtment environment was IntelliJ.
